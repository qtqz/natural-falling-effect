# 自然飘落效果组件

## 介绍

这是一个让你的页面可以飘落花瓣，飘落树叶，以及下雨下雪的vue组件，拥有大量自定义选项。也支持直接以js模块形式使用（GUI界面仅vue可用）。

* 漂亮的效果
* 丰富的可配置项
* （如果使用vue）访客可以自定义配置
* 即使不使用vue，也可以作为单独的js文件引入
* GUI适配暗黑模式
* 性能较好
